# Assistant - n8n Workflow Automation

This repository contains an n8n installation for workflow automation and AI assistant capabilities.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
The project dependencies are already installed. To start n8n:

```bash
npm start
```

This will start the n8n server, typically accessible at `http://localhost:5678`.

### Features
- Workflow automation
- AI-powered assistant capabilities
- Integration with various services and APIs
- Visual workflow builder

### Usage
1. Start the n8n server with `npm start`
2. Open your browser and navigate to `http://localhost:5678`
3. Create your first workflow using the visual editor
4. Configure nodes and connections to automate your processes
